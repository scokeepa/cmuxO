# /cmux-help — 명령어 도움말

입력: `$ARGUMENTS`

---

## 라우팅

### 빈 입력 → 도움말 출력

```
═══════════════════════════════════════════════════════
  cmux AI 다중 자동 협업 플랫폼
═══════════════════════════════════════════════════════

  시작:
  ─────────────────────────────────────────────────────
  /cmux-start              오케스트레이션 시작 (이것만!)

  설정:
  ─────────────────────────────────────────────────────
  /cmux-config              AI 프로파일 확인
  /cmux-config detect       설치된 AI 자동 감지
  /cmux-config add <이름>   AI 추가 (커스텀도 가능)
  /cmux-config remove <이름> AI 제거

  운영:
  ─────────────────────────────────────────────────────
  /cmux-stop                오케스트레이션 종료
  /cmux-watcher-mute        와쳐 알림 토글 (on/off/status)

  긴급:
  ─────────────────────────────────────────────────────
  /cmux-pause               전체 일시 중지
  /cmux-pause resume        재개

  도움말:
  ─────────────────────────────────────────────────────
  /cmux-help                이 도움말 보기
  /cmux-help off            자동 도움말 끄기
  /cmux-help on             자동 도움말 켜기

  메모리:
  ─────────────────────────────────────────────────────
  agent-memory.sh drain     저널 → 영구 메모리 통합
  agent-memory.sh query X   메모리 검색
  agent-memory.sh stats     통계

  제거:
  ─────────────────────────────────────────────────────
  /cmux-uninstall           완전 제거 + 설정 롤백

  전제 조건:
  ─────────────────────────────────────────────────────
  cmux 0.62+ (cmux 터미널 세션 내에서 실행)
  Claude Code 2.1+
  Python 3.9+

  구조:
  ─────────────────────────────────────────────────────
  사용자(사장) ──→ Main(COO) ──→ 팀장 ──→ 팀원
  Watcher(감사실): 모니터링 + PC 리소스 관리

  개별 작업:
  ─────────────────────────────────────────────────────
  /cmux-start 실행 전에는 일반 Claude Code로 사용.
  Hook 간섭 없음.

═══════════════════════════════════════════════════════
```

### `off` → 자동 도움말 끄기

```bash
touch /tmp/cmux-help-disabled
```
→ "도움말 자동 출력을 껐습니다. /cmux-help로 언제든 확인 가능."

### `on` → 자동 도움말 켜기

```bash
rm -f /tmp/cmux-help-disabled
```
→ "도움말 자동 출력을 켰습니다."
